# Auto Insurance Claim Risk Analysis

## Overview

This project analyzes driver demographics and behavioral factors to identify key contributors to insurance claim risk.

Using Power BI, an interactive dashboard was built to visualize claim trends across driver age groups, driving experience levels, income brackets, and annual mileage.

## Data Preparation

The dataset was cleaned and structured using Microsoft Excel before being imported into Power BI for analysis and visualization.

## Tools Used

- Power BI
- Microsoft Excel
- Data Visualization
- Exploratory Data Analysis

## Key Insights

- Drivers aged **16–25** show the highest claim frequency (~72%).
- Claim probability **decreases as driving experience increases**.
- Drivers with **lower income levels show higher claim rates**.
- **Higher annual mileage correlates with higher claim risk**.
